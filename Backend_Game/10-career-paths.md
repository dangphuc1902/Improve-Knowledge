## Hướng A: Senior Game Server Engineer

### Lộ trình kỹ năng (thêm 1-2 năm)

```
Middle Game Backend (bạn sẽ ở đây)
    │
    ├── Deep C++ (templates, allocators, SIMD)
    ├── Game networking (netcode, prediction, lag compensation)
    ├── Game engine integration (Unreal Dedicated Server, custom engine)
    ├── Distributed game architecture (world partitioning, zone servers)
    ├── Database optimization cho game (player save, leaderboard at scale)
    └── DevOps for games (deploy, scaling, monitoring game-specific metrics)
    │
    ▼
Senior Game Server Engineer
    │
    ├── Lead small team (2-4 devs)
    ├── Design game server architecture from scratch
    ├── Performance budget planning
    ├── Cross-team collaboration (client, art, design)
    └── Live ops (hotfix, content update, A/B testing)
```

### Companies (VN & SEA)
- VNG, Garena/SEA, Amanotes, Topebox
- Riot Games, Blizzard, miHoYo (Remote/Singapore)
- Indie studios cần C++ server developer

### Ưu điểm
- ✅ Leverage thế mạnh C++ hiện tại
- ✅ Ít cạnh tranh (niche skill)
- ✅ Passion-driven career
- ✅ Technical depth cực cao

### Nhược điểm
- ❌ Ít vị trí tuyển dụng
- ❌ Game industry volatility (layoffs cycles)
- ❌ Crunch culture
- ❌ Khó remote (thường cần on-site)
- ❌ Lương thấp hơn fintech/FAANG

## Hướng B: General Backend / Software Engineer

### Lộ trình kỹ năng (thêm 6-12 tháng)

```
Middle Game Backend (bạn sẽ ở đây)
    │
    ├── Learn Go/Java/Python (1-2 tháng)
    ├── Cloud (AWS/GCP basics: EC2, S3, RDS, SQS)
    ├── Microservices patterns
    ├── REST API / GraphQL
    ├── CI/CD & DevOps maturity
    └── System design interview prep
    │
    ▼
Backend Software Engineer (General)
    │
    ├── Fintech, E-commerce, SaaS, Platform
    ├── Cloud-native development
    ├── Data engineering pipeline
    └── Eventually: Staff/Principal Engineer
```

### Transferable Skills (đã có)
- ✅ **C++ performance mindset** → valuable trong high-performance systems
- ✅ **Server-authoritative** → same pattern trong banking (trust nothing from client)
- ✅ **Event-driven** → Kafka/event sourcing rất hot
- ✅ **RPC** → gRPC standard trong microservices
- ✅ **Concurrency** → highly valued skill

### Companies (VN & Remote)
- **Fintech**: MoMo, VNPay, Tiki, Shopee
- **Tech**: FPT, VNG (non-game), Grab, Gojek
- **Remote**: Toptal, Remote.com, AngelList startups
- **FAANG**: Google, Meta, Amazon (SDE roles)

### Ưu điểm
- ✅ Nhiều cơ hội việc làm
- ✅ Lương cao hơn (đặc biệt fintech, FAANG)
- ✅ Remote-friendly
- ✅ Stable industry
- ✅ Clear career ladder (SDE I → II → Senior → Staff → Principal)

### Nhược điểm
- ❌ Cần học thêm cloud/new languages
- ❌ Cạnh tranh cao hơn
- ❌ Có thể bớt exciting nếu yêu game
