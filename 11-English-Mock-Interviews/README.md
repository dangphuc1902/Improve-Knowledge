# 🎯 English Mock Interview — Spaced Repetition System (SRS)

> **Goal**: Master English for Java Backend technical interviews through daily practice
> **Method**: Spaced Repetition — review cards at increasing intervals (1→3→7→14→30 days)
> **Time**: 20-30 minutes per day

## 🃏 How to Use SRS Cards

Each session file contains **flashcard-style Q&A pairs**.

**Practice method:**
1. Read the QUESTION
2. Say your answer OUT LOUD (don't write — you need to speak!)
3. Check the model answer
4. Rate yourself: ✅ Good | ⚠️ Okay | ❌ Need practice
5. Schedule next review based on rating

**Rating → Next Review:**
- ✅ Good → Review in 7 days
- ⚠️ Okay → Review in 3 days  
- ❌ Need practice → Review tomorrow
