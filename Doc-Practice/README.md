# TÀI LIỆU ĐÀO TẠO & LUYỆN PHỎNG VẤN SENIOR JAVA GAME BACKEND

Chào mừng bạn đến với lộ trình đào tạo chuyên sâu dành cho Java Game Backend Engineer. Bộ tài liệu này được thiết kế để đưa bạn từ cấp độ Junior lên Mid/Senior với tư duy thực chiến nhất.

## 📁 Danh sách các bài học

1.  [**PHẦN 1: JAVA CORE (NỀN TẢNG)**](Part1_JavaCore.md)
    - OOP trong Game, JVM Memory Model, GC Tuning, Collections.
2.  [**PHẦN 2: MULTI-THREADING (CỰC QUAN TRỌNG)**](Part2_MultiThreading.md)
    - Race condition, Deadlock, Locks, Tick Game Loop, 10k CCU.
3.  [**PHẦN 3: NETWORK PROGRAMMING**](Part3_Network.md)
    - TCP vs UDP, WebSocket, Protobuf, Latency synchronization.
4.  [**PHẦN 4: MICROSERVICES & SYSTEM DESIGN**](Part4_Microservices.md)
    - Service breakdown, API Design (gRPC vs REST), Game Flow.
5.  [**PHẦN 5: KAFKA (EVENT-DRIVEN)**](Part5_Kafka.md)
    - Producer/Consumer, Decoupling, Quest/Reward Case Study.
6.  [**PHẦN 6: REDIS (CACHE & REALTIME)**](Part6_Redis.md)
    - Data structures, Leaderboard, Session management.
7.  [**PHẦN 7: MONGODB / DATABASE DESIGN**](Part7_Database.md)
    - SQL vs NoSQL, Inventory/Player Data Schema design.
8.  [**PHẦN 8: KUBERNETES & CLOUD**](Part8_K8s_Cloud.md)
    - Docker, K8s, Agones (Stateful), Auto-scaling.
9.  [**PHẦN 9: SYSTEM DESIGN (LEVEL SENIOR)**](Part9_SystemDesign_Senior.md)
    - 1M CCU, Global Chat, Advanced Matchmaking.
10. [**PHẦN 10: GOLANG & KOTLIN (BONUS)**](Part10_Golang_Kotlin.md)
    - So sánh với Java, Khi nào nên dùng Go/Kotlin.
11. [**PHẦN 11: LỘ TRÌNH HỌC (ROADMAP)**](Part11_Roadmap.md)
    - Lộ trình 3 tháng - 6 tháng - 1 năm.
12. [**PHẦN 12: MOCK INTERVIEW**](Part12_MockInterview.md)
    - 60 câu hỏi phỏng vấn Junior -> Senior kèm mindset.
